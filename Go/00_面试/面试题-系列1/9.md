# 1. 9

## 1.1. 问题

### 1.1.1. 问题1

下面这段代码能通过编译吗？请简要说明。

```go
func main() {
    m := make(map[string]int)
    m["foo"]++
    fmt.Println(m["foo"])
}
```

### 1.1.2. 问题2

下面的代码输出什么，请简要说明？

```go
func Foo() error {
    var err *os.PathError = nil
    // …
    return err
}

func main() {
    err := Foo()
    fmt.Println(err)
    fmt.Println(err == nil)
}
```

## 1.2. 答案

### 1.2.1. 答案1

能通过编译。

上面的代码可以理解成：

```go
func main() {
    m := make(map[string]int)
    v := m["foo"]
    v++
    m["foo"] = v
    fmt.Println(m["foo"])
}
```

### 1.2.2. 答案2

参考答案及解析：nil false。

知识点：接口值与 nil 值。**只有在值和动态类型都为 nil 的情况下，接口值才为 nil。Foo() 函数返回的 err 变量，值为 nil、动态类型为 `*os.PathError`，与 nil（值为 nil，动态类型为 nil）显然是不相等**。我们可以打印下变量 err 的详情：


```go
fmt.Printf("%#v\n",err)   // (*os.PathError)(nil)
```

一个更合适的解决办法：

```go
func Foo() (err error) {
    // …
    return
}

func main() {
    err := Foo()
    fmt.Println(err)
    fmt.Println(err == nil)
}
```
